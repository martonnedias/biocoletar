# biocoletar

Sistema de gestão e coleta de óleo de cozinha usado. Especialistas em sustentabilidade em Teresina e região.

## Tecnologias
- React
- TypeScript
- Tailwind CSS
- Gemini AI (Assistente Virtual)
- Vite

## Como publicar
Este projeto está configurado para deploy automático via GitHub Actions. Ao fazer um push para a branch `main`, o site será publicado no GitHub Pages.
